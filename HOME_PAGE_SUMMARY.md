## 🎨 Color Customization

### To Change Green Color
Find in `style.css`:
```css
--brand-900: #0a1f1b;  ← Change background
--brand-700: #1a5c52;  ← Change primary
--brand-500: #2faa6b;  ← Change accent
```

### To Change Orange Accents
```css
--accent-500: #f59f4a;  ← Change badges
--accent-600: #e3832c;  ← Change hover
```
