## 🎬 Animation Effects

```
ENTRANCE ANIMATIONS:
├─ fadeInUp: Slides up while fading in (0.8s)
├─ fadeInDown: Slides down while fading in (0.6s)
├─ slideInRight: Slides from right (0.6s)
├─ bounceIn: Bouncy entrance (0.6s)
└─ dropdownSlide: Classic dropdown effect (0.3s)

CONTINUOUS ANIMATIONS:
├─ float: Gentle up/down floating (3s loop)
├─ drift: Decorative blob drifting (8-10s loop)
├─ shimmer: Shine effect across surface (3s loop)
├─ pulse: Opacity pulsing (2s loop)
└─ glow: Shadow pulsing (2s loop)

HOVER ANIMATIONS:
├─ translateY(-16px): Card lift effect
├─ scale(1.02-1.15): Subtle zoom effects
├─ rotate(8-15°): Gentle rotations
├─ transform + box-shadow: Combined effect
└─ All with cubic-bezier(0.34, 1.56, 0.64, 1)
```
